# actividad_iv
Actividad iv

Ejercicio: Formulario + Envío por URL + Lectura de Parámetros

Fecha de entrega: 17 de Junio - 18hs.
Incluir el link de netlify y el repositorio en el foro.

Objetivo:
Crear una aplicación simple en JavaScript donde:

Desarrollador A: construye un formulario, (nombre - 5 caracteres, email - formato valido, edad - 18 a 60, pais - agregar opciones, acepta terminos) valida los datos y los envía mediante parámetros en la URL.
Desarrollador B: recibe esos parámetros en una nueva página y muestra la información al usuario.

Tanto el desarrollador A y B deben realizar los commits correctamente.
